# About

A simple lib to import Tiled maps to Heaps projects. It only supports basic stuff, and it's not meant to support 100% of all Tiled features. You can extend it quite easily :)

# Usage

See Test.hx

```bash
│   Test.hx
│
└───tiled
    │   TiledMap.hx
    │
    └───com

            TTileset.hx
```
